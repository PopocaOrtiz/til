# vim
## tabs
- `:tabedit`
- `:tabs`
- `:tabn` next tab or `gt` in normal mode
- `:tabp` previous tab or `gT` in normal mode

- :sp split vertically
- :vs split horizontal

## buffers
- :ls list all buffers
- :b switch to an open buffer

## bookmarks
- `m{letter}` set the bookmark with the {letter} identifier
- `'{letter}` go to the {letter} bookmark

## miscelaneous
- `%` current file path

## replace
- `:%s/foo/bar/gc`
